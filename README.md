# renyi
基于vue.js的webapp   
使用框架 : vue.js , hbuilder ,bootstrap (为了这个栅格化)<br>
主要功能 ： 1. 登录/注册<br>
           2. 发布任务 -> 上传图片   <br>
           3. 抢单 <br>
           4. 查看任务详情<br>
           5. 任务通知 -> 1.正在进行中 -> 如果是自己发布被抢单的任务则可以确认完成<br>
                         2.已完成<br>
           6.个人中心 ->  1. 修改头像<br>
                         2. 修改用户名<br>
                         3. 退出<br>
